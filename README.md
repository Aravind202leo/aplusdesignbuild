## Files

- `index.html` - Page structure and content
- `styles.css` - Responsive visual design
- `script.js` - Mobile menu, theme toggle, filters, and form feedback
- `site-config.js` - Editable phone, email, map, and social links

## Run

Open `index.html` in a browser. The site uses remote image assets from Unsplash and Lucide icons from a CDN, so an internet connection is needed for the full visual experience.

## Updating Links

Edit `site-config.js` to change phone numbers, email, Google Maps URL, Instagram URL, or to add future Facebook/X links. Leave a social `href` empty to keep it as a disabled placeholder.
